
## Commit changes

Add single item

``` bash
git add filename.extension # e.g. git add app.js
```

Add all files

```bash
git add .
```

Tag

```bash
git tag <tagname> # e.g. git tag v2.1.0
```

Commit

```bash
git commit -m "My commit message"
```

Commit and Add together 

```bash
git commit -am "Commit message"
```

Pull code

```bash
git pull
```

Push code

```bash
git push
```
